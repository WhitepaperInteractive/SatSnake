# Classic Snake Game

A faithful recreation of the classic Snake game, built with HTML5 Canvas and vanilla JavaScript.

## Features
- Classic snake gameplay
- Score tracking
- Game over on wall/self collision
- Retro pixel-perfect style
- Mobile-friendly (use arrow keys)

## How to Play
- Use arrow keys to control the snake
- Eat the red food to grow
- Don't hit the walls or yourself!
